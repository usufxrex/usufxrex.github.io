CACHING & PERFORMANCE
===============================================

Cache Duration: 24 hours
- First load: Fetches from GitHub API
- Subsequent loads: Uses cached data (24h)
- After 24h: Auto-refreshes from API

Cache Storage: Browser localStorage
Cache Key: 'github_repos_cache'

To manually refresh:
1. Press F12 (open console)
2. Type: localStorage.removeItem('github_repos_cache');
3. Refresh the page

===============================================
TROUBLESHOOTING
===============================================

If GitHub projects don't load:
1. Check internet connection
2. Verify GitHub username is correct
3. Check browser console (F12) for errors
4. Clear cache (see above)
5. Check GitHub API rate limit:
   https://api.github.com/rate_limit

If showing wrong repos:
- Make sure repos are public (not private)
- Check if they're forks (forks are hidden)
- Verify repository descriptions on GitHub
